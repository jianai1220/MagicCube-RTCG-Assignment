# MagicCube-RTCG-Assignment

This is an individual assignment of the course SECV3123 Real-Time Computer Graphics.

<br>

## Engine Used:

* Unity (6000.3.12f1)
* Vuforia Engine

<br>

## Assets Used:
* Low-Poly Simple Nature Pack (by JustCreate)
* Low-Poly Forest Survival Starter Pack Lite (by DEVTRICKED)
* Low Poly Living Room Pack (by The PolyNestLab)
* Bird (by Zacxophone)
* Low Polygon Soccer Ball (by Ahmet Gencoglu)
* VFX URP - Fire Package (by Cartoon VFX by Wallcoeur)
* Quirky Series - FREE Animals Pack (by Omabuarts Studio)
